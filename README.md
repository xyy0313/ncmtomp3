# ncmdump

Node.js 版网易云音乐 `.ncm` 批量解密为 MP3。

## 使用步骤

1. 安装依赖

```bash
npm install
```

2. 将待转换的 `.ncm` 文件放入项目根目录下的 `ncm` 文件夹（仅放 `.ncm`，避免脚本误处理其他文件）。

3. 执行转换

```bash
node index.js
```

4. 转换结果写入 `mp3` 文件夹：与源文件同名的 `.mp3` 音频，以及同名的 `.jpg` 封面图。

## 目录说明

| 目录 | 用途 |
|------|------|
| `ncm/` | 输入：放置 `.ncm` 文件 |
| `mp3/` | 输出：生成的 `.mp3` 与 `.jpg` |
